Run FutureFit locally

AI resume generation is part of the full FutureFit app and requires the Django backend. To use this feature, run FutureFit with Docker:

git clone https://github.com/LaneyCommits/FutureFit.git
cd FutureFit
docker compose up

Then open http://localhost:8000/resume/ and click AI-generated resume.
